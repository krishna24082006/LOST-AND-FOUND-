# Lost & Found Portal

A web app for Ambalika Institute of Technology & Management, Lucknow to report and search lost/found items.

## Features
- Report lost/found items
- Search by name/desc/location
- Browse all items
- Responsive UI

## Run locally
```bash
cd projectlostfound-main
py app.py
```
Open http://127.0.0.1:5000

## Tech
Flask + SQLite
